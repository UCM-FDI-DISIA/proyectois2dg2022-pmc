La documentación del proyecto se encuentra en la siguiente ruta:
./Documentos/FINAL/DocumentoOficial.pdf

Una vez descargado el proyecto debes abrir eclipse y seleccionar
el Workspace que desee.
Primero te aparecerá una ventana de bienvenida que hay que
cerrar dándole a la esquina superior derecha. A continuación pulsa:
	-File, situado en la esquina superior izquierda.
	-New
	-Java Project
	-Deselecciona "Use default location"
	-Browse... y selecciona la carpeta Rolit
	-Finish
Si todo salió bien debería poder ver la carpeta Rolit a mano
izquierda. Ábrela y pulsa botón derecho sobre la clase Rolit,
situada en el paquete model, dentro de la carpeta src. Pulse:
	-Run as...
	-1 Java application
Siga las indicaciones que le aparecen en la consola:
Choose mode: 
1. Console Mode
2. GUI Mode

En el caso de seleccionar console debe seguir las instrucciones de
consola. Por otro lado, si elige GUI, aparte de las indicaciones 
visuales, tendrá acceso a un tutorial que le explicará más 
detalladamente qué hacer. En caso de no poder ejecutar el tutorial
en Java 8, pruebe a ejecutarlo desde Java 11 (se usan librerías JavaFX, que en ocasiones
generan problemas con las versiones de Java).
